# P31 CLI Site

Source for [cli.p31ca.org](https://cli.p31ca.org) — landing page, documentation, and installer for the P31 sovereign AI CLI.

## Pages

- `/` — Landing page (Kilo Code style hero, badges, install, 6 feature cards)
- `/install` — One-line installer script
- `/docs/` — Documentation hub (6 guide pages)
- `/downloads/` — Cross-platform CLI binaries

## Deploy

```bash
npx wrangler pages deploy . --project-name p31-cli
```
